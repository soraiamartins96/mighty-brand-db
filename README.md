# Mighty Brand DB — Foundation

Static reference page for Mighty's brand foundation: colours, fonts, buttons, and logos.

## Run it

Open `foundation.html` in a browser, or serve the folder locally:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000/foundation.html
```

## Files

- `foundation.html` — the page
- `_shared.css` — shared styles
- `_database.js` — brand data (`window.__MIGHTY_DB`)
- `_nav.js` — top nav bar
- `_rules-footer.js` — per-page rules footer
- `foundations/assets/` — logo SVGs and PNGs
